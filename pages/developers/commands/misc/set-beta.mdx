# Set Beta Developer Command

This command will allow developers of the bot to set a server as a beta tester server. This will allow the server to use beta commands and features.

## Usage

```
set-beta
```

## Examples

```
set-beta
```

**[bot]: Set this server to beta**

**[bot]: Successfully set beta commands for guild `Example Guild`**
