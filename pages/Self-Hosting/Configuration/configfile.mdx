# Config File

The configuration file allows you to customize various aspects of the system. Below is an example of how to configure the file:

| Variable                  | Description                                | Type     |
|---------------------------|--------------------------------------------|----------|
| prefix                    | Prefix for commands                        | string   |
| link.name                 | Name of the link command                    | string   |
| link.id                   | ID of the link command                      | string   |
| catch.name                | Name of the catch command                   | string   |
| catch.id                  | ID of the catch command                     | string   |
| help.name                 | Name of the help command                    | string   |
| help.id                   | ID of the help command                      | string   |
| TIME_TO_DELETE            | Time in seconds to delete messages          | number   |
| SHOP_EXPIRE_DAYS          | Number of days for shop item expiration     | number   |
| RATE_LIMIT_WINDOW         | Time window in seconds for rate limiting    | number   |
| NO_RATE_LIMIT             | Array of commands exempt from rate limiting | string[] |
| VERSION                   | Version of the system                       | string   |
| allow_developer_applications | Flag to allow developer applications       | boolean  |
| DEVELOPER_PREFIX          | Prefix for developer commands               | string   |
| COMMUNITY_UPDATES_CHANNEL | Channel for community updates               | string   |
| IS_IN_DEV_MODE            | Flag to indicate development mode           | boolean  |
| BOT_ID                    | ID of the bot                               | string   |
| COINS.unverified          | Number of unverified coins                  | number   |
| COINS.verified            | Number of verified coins                    | number   |
| OPEN_FOR_DEVLOPERS        | Flag to indicate if open for developers     | boolean  |
| AUTOMATED_USERS           | Array of automated user IDs                 | string[] |
