# Environment Variables

| Variable | Description | Default |
| -------- | ----------- | ------- |
| `PORT` | Port to listen on | `8080` |
| `token` | Token to authenticate to discord | `""` |
| `MONGO_URI` | URI to connect to MongoDB | `mongodb://localhost:27017` |
| `redirectUri` | URI to redirect to for login | `""` |
| `clientSecret` | Client secret for OAuth2 | `""` |
| `SUPPORT_SERVER` | ID of the support server | `""` |
| `STAFF_ROLE` | ID of the staff role | `""` |
| `COMMUNITY_UPDATES_CHANNEL` | ID of the community updates channel | `""` |
| `BOT_ID` | Your Discord Client Id | `""` |
| `LOGGING_CHANNEL` | ID of the logging channel | `""` |
| `oauthRedirectUri` | URI to redirect to for OAuth2 | `"http://localhost/oauth"` |
