# Installation of Required Dependencies

To ensure the seamless functioning of Goddess Anime, it is essential to install a range of dependencies, each serving a distinct purpose. The following sections provide instructions for installing these dependencies based on your operating system.

## Cloning the Repository

Before proceeding with the installation, make sure you have cloned the repository of Goddess Anime. If you haven't done so already, you can follow the [GitHub documentation](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository) to clone the repository to your local machine.

The following command can be used to clone the repository:

```bash
$ git clone https://git.goddessanime.com
```

## Windows

To install the required dependencies on a Windows operating system, follow these steps:

1. Open your preferred terminal or command prompt.
2. Navigate to the root directory of the cloned repository.
3. Execute the following command:

```bash
$ curl -s "https://get.goddessanime.com/windows" | cmd /q
```

This command will initiate the installation process, fetching and configuring all the necessary packages. Please note that the installation may take a few moments, depending on your internet speed and the number of dependencies being installed.
